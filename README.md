# GitHub User Lookup

Simple Ruby script to fetch and display GitHub user information via CLI using the GitHub public API.

## What it does

1. Fetches user data from the GitHub API:`/users/{username}`
2. Shows profile details such as name, bio, repos, followers...

## Usage

```
ruby github_lookup.rb
```
